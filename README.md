# Journal Homepage (HTML & CSS)

This project is a simple, clean, and responsive homepage built using HTML and CSS. It focuses on semantic HTML5 structure, modular CSS, and a layout suitable for modern websites.

---

## Features

- Semantic HTML5 tags (header, main, section, footer)
- Responsive layout using Flexbox and media queries
- Reusable utility classes for spacing and alignment
- Navigation bar with logo section
- Hero/intro section
- Category or department bar
- Footer section
